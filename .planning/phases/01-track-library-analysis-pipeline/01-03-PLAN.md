From Plan 02: API endpoints:
- POST /api/tracks/upload (multipart) -> 201 [{id, ...track}]
- GET /api/tracks?projectId={id} -> Track[]
- GET /api/tracks/{id}/status -> {analysisStatus, analysisResult}
- GET /api/tracks/{id}/audio -> audio stream
- GET /api/tracks/{id}/peaks -> peaks JSON
- PUT /api/tracks/{id} -> update track metadata
- DELETE /api/tracks/{id} -> remove track
</interfaces>
